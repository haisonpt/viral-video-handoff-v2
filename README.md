# Viral Video Standalone App

Standalone app for viral video workflow, independent from PostNhanh AI.

## Local run

```bash
npm install
npm run build
npm run server
```

In another terminal:

```bash
npm run dev
```

## Environment
Create `.env.local` in this folder:

```env
RUNWAY_API_KEY=your_runway_api_key_here
```

## Open first
- `docs/VIRAL_VIDEO_COMPLETE_GUIDE.md`
- `docs/VIRAL_VIDEO_QUICKSTART.md`
- `docs/VIRAL_VIDEO_HANDOFF.md`

## Notes
- Prompt video: English
- Voice/dialogue: Vietnamese
- Character + voice continuity for serial workflows
- Runway is the primary real provider
